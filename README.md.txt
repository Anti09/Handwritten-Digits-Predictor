This Project is useful for Recognize the HANDWRITTEN DIGITS.

User just have to DRAW the Digit and then Hit the Predict Button, It will Predict the drawn Digit.

This is a MACHINE LEARNING BASED PROJECT, I used KNN-ALGORITHM because this is the only algorithm which is BEST fit for HANDWRITTEN_DIGIITS.

Once Run it will open a window/form or the TKINTER WINDOW, in which all the OPERATION will be perform.
Operations like :-
1>    SAVE
2>    PREDICT
3>    CLEAR
4>    EXIT
